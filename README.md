# DonnaGa

둘이 함께 쓰는 PWA 가계부 초안입니다. 위플 가계부 스타일의 빠른 입력, 월간 요약,
일간·주간·월간 내역 보기, 카테고리 통계, 예산, 자산 현황을 우선 구현했습니다.

## 현재 포함된 기능

- 설치 가능한 PWA 구조
- 수입, 지출, 이체 기록
- 사용자 구분: `나`, `예비신부`
- 월별 합계와 잔액 카드
- 일간, 주간, 월간 그룹 보기
- 카테고리별 지출 통계
- 예산 사용량 표시
- 자산 계정별 월 반영 잔액
- 브라우저 `localStorage` 저장
- JSON 내보내기

## 실행 방법

정적 파일만으로 구성되어 있어 로컬 서버로 열면 됩니다.

```bash
python3 -m http.server 4173
```

그 다음 브라우저에서 `http://localhost:4173` 로 접속합니다.
