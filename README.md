# DonnaGa (돈나가)

정우와 솔이, 둘만의 프라이빗 커플 가계부입니다.
위플 가계부의 직관적이고 빠른 입력 방식을 차용하여 모바일 웹(PWA) 환경에 완벽히 최적화했습니다.

## 📱 화면 미리보기
*(여기에 실제 앱 스크린샷 이미지를 추가하세요)*

## ✨ 핵심 기능
- **오프라인 우선 (Local-first):** 지하철, 엘리베이터 등 네트워크가 끊겨도 언제든 1초 만에 바로 입력 가능.
- **빠른 동기화:** 내가 쓴 내역이 짝꿍 폰에도 반영되어 실시간 지출 내역 공유.
- **5대 예산 체계:** 생활 지출, 고정 지출, 변동 지출, 저축, 투자로 돈의 흐름을 한눈에 관리.
- **프라이빗 보안 (PIN Auth):** 둘만 아는 4자리 PIN 번호 인증으로 외부인의 접근과 수정을 철저히 차단.
- **네이티브 앱 경험 (PWA):** 브라우저 주소창 없이 바탕화면 아이콘으로 진짜 앱처럼 실행되며, 업데이트 배포 시 시각적 알림(Red Dot) 제공.

## 🛠 기술 스택 & 아키텍처
- **Frontend:** Vanilla JavaScript, HTML, CSS (가볍고 빠른 네이티브 렌더링)
- **Local DB:** IndexedDB (Dexie.js 기반의 강력한 브라우저 내장 저장소)
- **Cloud/Backend:** Cloudflare Pages & D1 (Serverless SQLite)
- **Design/Icons:** Lucide Icons 기반 커스텀 5-Tab UI

## 🚀 아키텍처 동작 흐름
1. **입력:** 앱에서 내역을 추가하면 먼저 내 폰의 로컬 DB(IndexedDB)에 즉시 저장되어 화면에 바로 나타납니다.
2. **동기화(Push):** 백그라운드에서 Cloudflare Functions API를 통해 D1 데이터베이스로 변경 사항을 안전하게 전송합니다. (인증된 PIN 동봉)
3. **동기화(Pull):** 앱을 켜거나 새로고침 시 서버에서 새로운 내역을 가져와 상대방의 입력 사항을 내 폰에 업데이트합니다.
