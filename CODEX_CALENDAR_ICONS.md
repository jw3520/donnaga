# 달력 UI 고도화 지시서: 이벤트 미니 아이콘 도입

달력(`calendar-grid`)의 날짜 칸에 숫자가 너무 많아지는 것을 방지하고 가독성을 높이기 위해, 지출 외의 이벤트(수입, 저축, 투자)는 금액 대신 직관적인 **'미니 아이콘'**으로 표시합니다.

## 1. 렌더링 로직 수정 (`app.js` - `renderCalendar`)
- **지출(Expense):** 기존처럼 날짜 아래에 금액(예: `-15,000`)을 숫자로 텍스트 표시합니다.
- **수입/저축/투자 이벤트 감지:** 
  - 해당 일자의 트랜잭션 중 `type`이 `income`이거나, `category`가 '저축' 또는 '투자' 그룹에 속하는 내역이 있는지 판별하세요.
- **아이콘 HTML 삽입:**
  - 이벤트가 감지되면 날짜 셀(`.calendar-day`) 내부에 미니 아이콘용 HTML(Lucide icon)을 주입하세요.
  - 하루에 여러 종류의 이벤트가 있다면 아이콘을 가로로 나란히(최대 2~3개) 배치합니다.

## 2. 아이콘 매핑 및 디자인 (`styles.css` & `app.js`)
- **수입 (Income):** `<i data-lucide="coins"></i>` (색상: `var(--income)` 또는 민트색)
- **저축 (Savings):** `<i data-lucide="piggy-bank"></i>` (색상: `#6eaaf5` 등 연파랑)
- **투자 (Investment):** `<i data-lucide="trending-up"></i>` (색상: `#a881af` 등 보라색)

## 3. UI/CSS 스타일링 가이드 (`styles.css`)
- **위치:** 미니 아이콘은 날짜 숫자(`.calendar-day__date`)의 우측 상단, 혹은 날짜와 같은 줄에 아주 작고 앙증맞게 배치하세요. (`position: absolute`를 활용하여 우측 상단 모서리에 띄우는 것을 추천합니다.)
- **크기:** 아이콘(`svg`)의 너비와 높이를 `12px`, `stroke-width`를 `2` 정도로 설정하여 앙증맞고 뚜렷하게 보이게 하세요.
- **정렬:** 지출 금액 숫자의 레이아웃을 해치지 않아야 합니다.

## 4. 개발자 주의사항
- DOM에 아이콘 HTML을 동적으로 주입한 후에는 반드시 `lucide.createIcons()`를 다시 호출하여 렌더링되게 해야 합니다.
