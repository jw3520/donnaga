# UI 리뉴얼 작업 지시서: 메인 화면 상단바 및 탭바 (위플 가계부 스타일)

이 문서는 AI 코딩 에이전트(Codex 등)가 `donnaga` 프로젝트의 UI를 위플 가계부 스타일로 리뉴얼하기 위한 구체적인 가이드입니다.

## 1. 목표 (Goal)
`index.html`과 `styles.css`를 수정하여 텍스트 위주의 투박한 UI를 위플 가계부 특유의 깔끔하고 직관적인 디자인으로 변경합니다.

## 2. 디자인 시스템 규칙 (Design System)
- **메인 컬러 (Primary):** 위플의 민트색 (`#4DB6AC` 또는 `#52c4b9`)
- **지출 컬러 (Expense):** 연한 빨간색/분홍색 (`#e57373` 또는 `#ff6b6b`)
- **수입 컬러 (Income):** 민트색/초록색 (`#4DB6AC` 또는 `#4caf50`)
- **아이콘 라이브러리:** 텍스트 기호(▥, ☰)를 모두 제거하고, **Lucide Icons** (CDN 사용) 또는 깔끔한 SVG 라인 아이콘으로 전면 교체합니다.

## 3. 세부 구현 요구사항 (Tasks)

### Task 1: 아이콘 라이브러리 추가
`index.html`의 `<head>`에 Lucide 아이콘 스크립트를 추가하고, `app.js` 하단에서 `lucide.createIcons()`를 호출하도록 설정하세요.
- CDN: `<script src="https://unpkg.com/lucide@latest"></script>`

### Task 2: 상단바 (Topbar) 리뉴얼
- **배경색:** 위플 스타일의 민트색 (`#4DB6AC`) 적용. 텍스트와 아이콘은 흰색(`#ffffff`)으로 변경.
- **레이아웃 구조 변경:**
  - **좌측 영역:** [막대그래프 아이콘(통계)], [문서 아이콘(내역)] 배치
  - **중앙 영역:** 기존 `2026년 3월` 텍스트를 `10월 ∨` (현재 월 + Chevron Down 아이콘) 형태로 변경하여 날짜 선택의 느낌 강조.
  - **우측 영역:** [돋보기 아이콘(검색)], [설정(톱니바퀴) 아이콘] 배치

### Task 3: 하단 탭바 (Tabbar) 리뉴얼
- 기존 4개의 텍스트 탭을 위플 스타일의 **5개 아이콘 탭**으로 변경합니다.
- **탭 구성 (좌측부터):**
  1. **달력:** 캘린더 아이콘 (`data-screen-target="calendar"`)
  2. **가계부(리스트):** 문서/리스트 아이콘 (`data-screen-target="list"`) *※ 필요시 list screen 추가*
  3. **차트:** 파이 차트 아이콘 (`data-screen-target="stats"`)
  4. **예산:** 돼지저금통 또는 지갑 아이콘 (`data-screen-target="budget"`)
  5. **설정:** 톱니바퀴 아이콘 (`data-screen-target="settings"`)
- 탭바의 텍스트는 제거하거나 아주 작게 아이콘 아래에 배치하고, 활성화된 탭은 민트색으로 강조되도록 CSS를 수정하세요.

### Task 4: 영웅 요약 (Hero Summary) 영역 다듬기
- 상단바 바로 아래에 위치하는 수입/지출 요약 영역의 폰트를 더 깔끔하게 다듬고, 수입은 민트색, 지출은 붉은색 텍스트로 명확히 대비되도록 CSS를 수정하세요.

## 4. 완료 조건
위 4가지 태스크를 반영하여 `index.html`과 `styles.css`를 업데이트한 후 결과를 보고하세요. 기존의 JavaScript 로직(데이터 렌더링 등)이 깨지지 않도록 DOM 구조 변경 시 `id`와 `class` 매핑에 주의하세요.
